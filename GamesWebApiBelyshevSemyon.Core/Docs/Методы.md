# Методы API

## Соревнования
***
### `/api/competitions`
#### Описание
Метод GET, позволяющий получить список всех соревнований в формате JSON.
#### Ожидаемые коды
- `200` — успешный запрос.
***
### `/api/competitions/{id}`
#### Описание
Метод GET, позволяющий получать конкретное соревнование по его идентификатору в формате JSON.
#### Ожидаемые коды
- `200` — успешный запрос;
- `404` — запрашиваемый ресурс не найден.
***
### `/api/competitions`
#### Описание
Метод POST, позволяющий добавлять новое соревнование.
#### Тело запроса
```json
{
  "Name": "<значение>",
  "Date": "<значение>",
  "Location": "<значение>",
  "SportType": "<значение>"
}
```
#### Ожидаемые коды
- `201` — ресурс успешно создан;
- `400` — неверные входные данные.
***
### `/api/competitions/{id}`
#### Описание
Метод PUT, позволяющий обновлять конкретное существующее соревнование по его идентификатору. Те поля, которые не заполнены, не будут менять значение
#### Тело запроса
```json
{
  "Name": "<значение>",
  "Date": "<значение>",
  "Location": "<значение>",
  "SportType": "<значение>"
}
```
#### Ожидаемые коды
- `201` — ресурс успешно создан;
- `404` — запрашиваемый ресурс не найден.
***
### `/api/competitions/{id}`
#### Описание
Метод DELETE, позволяющий мягко удалять конкретное существующее соревнование по его идентификатору
#### Ожидаемые коды
- `200` — успешный запрос;
- `404` — запрашиваемый ресурс не найден.
***
